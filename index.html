    About OpenYardage

    This is a web version of Hacker Yardage, a Python tool that generates yardage books from open data sets. (It can also add elevation and slope information.)

    All data is generously provided and maintained by the volunteers at OpenStreetMap. If you value this tool, please support them. If you see missing data, you can sign up to contribute it yourself.

    Export Yardage Book

    Images (ZIP)
    Every hole as a PNG image, bundled in a ZIP file. Useful for custom layouts, presentations, or importing into other apps.
    PDF — Digital
    One hole per page, optimized for on-screen viewing. Great for sharing, or for reading on a phone on the course.
    PDF — Printable
    Four holes per page in a compact layout. Print double-sided on the short edge, cut along the center line, stack and fold to make a pocket yardage book.
